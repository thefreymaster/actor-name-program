# Actor Name Parser

A program to parse string data into readable json objects

## Getting Started

### Install
Install the required dependencies
```
npm install
```

### Run Program
Run the program with the following script
```
npm run start
```

### Run Tests
Run tests for program
```
npm run test
```

### Expected Output
The expected output can be found in 
```
output.json
```# actor-name-program
